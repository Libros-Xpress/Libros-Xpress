Requisitos Técnicos – Libros/Xpress v2.0.1

Introducción
Este documento define los requisitos funcionales y no funcionales del sistema Libros/Xpress, una aplicación de escritorio desarrollada en Python con PySide6 para la gestión integral de librerías y papelerías. Describe qué debe hacer el sistema, cómo debe comportarse y las restricciones que debe cumplir para garantizar su correcto funcionamiento.

Descripción del Sistema Actual
Actualmente, muchas librerías y papelerías pequeñas y medianas operan con procesos manuales: el inventario se registra en cuadernos u hojas de cálculo, las ventas se documentan en libretas o tickets, y la facturación se realiza con formatos genéricos. No existe una integración entre la tienda física y un posible canal de ventas en línea, lo que provoca errores de stock, pérdida de tiempo y dificulta la generación de reportes para la toma de decisiones.
