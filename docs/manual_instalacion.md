# Manual de Instalación

## Requisitos del sistema
- Sistema operativo: Windows 10 o superior
- Python 3.12 instalado (si se ejecuta desde fuente)
- 4 GB de RAM recomendados
- 100 MB de espacio en disco

## Instalación desde ejecutable
1. Descargar `LibrosXpress.exe` de la carpeta `dist/`.
2. Copiar las carpetas `data/` y `assets/` junto al ejecutable.
3. Ejecutar `LibrosXpress.exe`.

## Instalación desde código fuente
1. Clonar el repositorio: `git clone <URL>`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar: `python src/main.py`
