# Ficha de Catalogación del Software

- **Nombre del sistema:** Libros/Xpress
- **Versión:** 1.5.0-beta
- **Descripción:** Sistema de gestión integral para librería y papelería que digitaliza ventas, inventario y facturación, con tienda en línea y programa de fidelización.
- **Tecnologías base:** Python 3.12, PySide6, JSON, PyInstaller
- **Autores:** Robert Cerón, David Solís, Juan Castro
- **Licencia:** Uso académico – Universidad (Ingeniería de Software III)
- **Dependencias:**
  - PySide6
  - PyInstaller
  - pytest, pytest-cov
  - pdoc
